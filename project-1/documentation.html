NoteWorthy:

This website has been designed to take data from the OpenDota API and present it in a usable and digestable manner for new players to get a basic understanding of each hero. The website utalizes Bulma to style the pages and make the information presentable as well as customized web components to make the pages look unique and easy to read. The website stores your last search terms within localized storage to provide easy use and extra functionality to the page. It also utalizes Firebase to track how many favorites each hero has.

Grading:

In this project I utalized fetch and promises to grab data from my API to be presented to the user. I extensivly utalized the Bulma framework to styalize my webpages and present the information on the screen to the user in an easy and digestable manner while also keeping a uniform color scheme and style between the pages. I created multiple custom web components that are used across a multitude of the web pages for more variety in styling and design. I also utalized Firebase to track how many favorites each hero has and presented that data to the users.
