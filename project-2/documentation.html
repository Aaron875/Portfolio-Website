NoteWorthy:

This website has been designed to create a 3D visual Lorenz Attractor with canvas 2D. The user is capable of messing with the X, Y, and Z values of the attractor using audio files that create a unique 3D image depending on what contols the user activates while the music is playing. The website utilizes bulma to make the pages aesthetically pleasing to the eye without sacraficing functionality. It also utilizes Fetch to load default control values from a JSON file.
